# SAP Services Deal Tracking

## Opportunity Stages (Standard Close Plan)
1. Lead / Identification — Prospecting & initial contact
2. Qualification — BANT/MEDDPICC validated, pain/impact confirmed, stakeholders mapped
3. Discovery & Needs Analysis — Deep dive workshops, value assessment, technical fit
4. Solution Development / Value Proposition — Proposal/SoW, demo/PoC, ROI/business case
5. Negotiation & Contracting — Commercial negotiation, legal/procurement review, executive alignment
6. Close — Signed contract/order, handover to delivery, win/loss review
