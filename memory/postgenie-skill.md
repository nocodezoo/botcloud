# PostGenie Automation Skill

## Overview
Automatically creates Instagram video posts on PostGenie using your logged-in Chrome browser.

## Prerequisites
1. User must be logged into PostGenie in Chrome
2. OpenClaw browser relay must be connected

## Usage
**Command:** "create postgenie post" or just "create an Instagram post"

## Process

### Step 1: Navigate to Create Page
- Navigate browser to: https://postgenie.app/create

### Step 2: Fill Post Content
- Fill textbox "Enter your post content..." with the user's prompt
- Default prompt: "Amazing sunset at the beach ðŸŒ… #nature #sunset #beautiful"

### Step 3: Set Colors & Styles (Optional)
- Click button "Colors & Styles"
- Set Font Size slider to 42 (or user's preference)
- Set Vertical Position slider to user's preference

### Step 4: Select Video from Collections
- Click button "View Media Collections"
- Select collection: "UGC - Influencers V2" (or user's choice)
- Click first/selected video thumbnail
- Click the download/select button (second icon) to confirm selection

### Step 5: Set Duration
- Set Video Duration slider to 10 seconds (or user's preference)

### Step 6: Enable Text Captions
- Check checkbox "Use Text Captions"

### Step 7: Add Post Caption
- Find and click the Post Caption section to expand it (look for "Post Caption" heading with down arrow or click the section)
- Fill textbox "Enter your Instagram caption..." with hashtags and caption
- Auto hashtags are enabled by default (5 hashtags)

### Step 8: Save Post
- Click button "Save Post" or "Generating Post..."
- Wait for processing to complete

## Important Notes
- The Post Caption section is at the bottom - click the heading/section to expand it before filling
- Wait for "Generating Post..." to complete before assuming success
- Browser relay must stay connected throughout

## Default Settings
- Video Duration: 10 seconds
- Font Size: 42px
- Use Text Captions: checked
- Auto Hashtags: 5
