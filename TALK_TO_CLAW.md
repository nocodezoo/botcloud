## Just Say What You Want

**Don't:** "Can you help me with something"
**Do:** "Write a Python script that does X"

---

## Best Formats

**For code/tasks:**
```
[Action] + [What] + [Details]

Example: "Write a function that checks if a list is sorted"
Example: "Find research papers on quantum computing, summarize in 2 sentences"
```

**For memory:**
```
"Remember: [fact]"
"Search memory for [topic]"
```

**For web:**
```
"Find [thing]" or "What's the best [tool] for [use case]?"
```
